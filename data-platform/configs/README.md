# data-platform\configs

**What this directory is for:**
This directory contains the `data-platform\configs` part of the Locofinder monorepo.

**What files live here and what each does:**
- `data_sources.yaml`: Starter module or configuration for this directory.

**How work in this directory is expected to be implemented:**
Implement small, testable modules with clear function/class boundaries and update tests/docs with each change.

**Inputs/Outputs:**
Inputs are source data, requests, or configs. Outputs are validated artifacts, API payloads, or build/runtime assets.
